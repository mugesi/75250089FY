# Getting Started with R - Part 1

R is a programming language used for statistical computing and graphics.
This guide will walk you through downloading and installing R, as well
as setting up RStudio.

## Step 1: Download R

R can be downloaded from the official CRAN (Comprehensive R Archive
Network) website:

-   [Download R](https://cran.r-project.org/)

1.  Click on the link above.
2.  Choose your operating system (Windows or macOS).
3.  Follow the installation instructions provided for your system.

## Step 2: Install R

### Windows Installation

1.  Download the `.exe` installer from CRAN.
2.  Run the installer and follow the on-screen instructions.
3.  Select the default options unless you have specific preferences when
    prompted.
4.  Complete the installation and open R.

### macOS Installation

1.  Download the `.pkg` file from CRAN.
2.  Open the file and follow the installation prompts.
3.  After installation, open R from Applications.

## Step 3: Install RStudio (Strongly Recommended)

RStudio is a software application that facilitates working more
efficiently with R.

-   [Download RStudio](https://posit.co/download/rstudio-desktop/)

1.  Choose the free desktop version.
2.  Download and install it following the prompts.
3.  Open RStudio and start coding in R!

## Step 4: Verify Installation

Open R or RStudio and type the following command to check your R
version:

    version

If the installation was successful, R should display its version
information.

## Step 5: Understanding RStudio Interface

RStudio consists of four main panes:

1.  **Source Pane** - This is where you write and edit scripts.
2.  **Console Pane** - Displays executed commands and their outputs.
3.  **Environment/History Pane** - Shows stored variables and command
    history.
4.  **Files/Plots/Packages/Help Pane** - Used for managing files,
    viewing plots, handling packages, and accessing help documentation.
